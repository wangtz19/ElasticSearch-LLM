# Elasticsearch LLM

## A QA Bot based on `elasticsearch`、`text2vec` and `LLM(ChatGLM2 etc)`

### Run
##### 1. install dependency
`pip install -r requirements.txt`
##### 2. run web ui
`python web_ui.py`
